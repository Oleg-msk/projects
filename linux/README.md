```
pwd - показывает где находишься

ls - показывает что находится в папке

ls -la - показывает что находится в папке, в удобном виде

cd /home/student/log - переход по абсолютному пути

cd .. - выход на уровень выше

cd ../.. - на 2 уровня выше

mkdir test - создает папку с названием «test» там, где вы находитесь

mkdir /home/test - создает папку «test» в директории /home, даже если вы 
сейчас не там

touch README.md - создаёт файл
```
